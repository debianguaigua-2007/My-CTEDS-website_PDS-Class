O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Ele é usado para estruturar o conteúdo na web, como textos, imagens, links e outros elementos, também é integravél com outras linguagens de programação, como CSS e JavaScript, e serve para manter uma estrutura para a sua página web.

Estrutura básica de um documento HTML

Um documento HTML é composto por várias tags que definem a estrutura e o conteúdo da página. As tags são formas de como definir os comandos que irão definir os elementos da página. Para criar uma página, primeiro é necessário um padrão de comandos, que é comum para cada site:

                        <!DOCTYPE html>
                        <html lang="pt-br">
                            <head>
                                <meta charset="UTF-8"/>
                                <meta name="viewport" content="width=device-width, initial-scale=1"/>
                                <title>Título da Página</title>
                            </head>
                            <body>
                                <header>
                                    <h1>Cabeçalho Principal</h1>
                                </header>
                                <main>
                                    <article>
                                        <h2>Título do Artigo</h2>
                                        <p>Conteúdo do artigo...</p>
                                    </article>
                                </main>
                                <footer>
                                    <p>Rodapé da Página</p>
                                </footer>
                            </body>
                        </html>

Esta é uma representação da parte mais crítica de uma página feita em HTML, sem estes componentes principais, a página não funcionaria corretamente. Cada tag tem uma função específica, como definir o cabeçalho, o corpo, os artigos, entre outros.

As tags são geralmente compostas por uma abertura e um fechamento, como <tag> e </tag>. Algumas tags podem ser auto-fechadas, como <img /> para imagens. É importante lembrar que as tags devem ser aninhadas corretamente para que o HTML seja válido.

Diferentes tags podem ter diferentes funções e portanto podem não caber em diferentes lugares, mas estão agrupados em categorias diferentes:

Tags de texto

São colocados dentro da tag <body>,são característicos para definir a estrutura de um texto:

  • <h1></h1> a <h6></h6> Indicam um titulo ou subtitulo para o texto, dependendo do numero colocado, o tamanho irá mudando
  • <p></p> (P de parágrafo) Define um parágrafo de texto
  • <strong></strong> Deixa o texto em negrito
  • <em></em> Deixa o texto em itálico
  • <article></article> Cria uma area de texto de tipo artigo

Tags de mídia

São colocados dentro da tag <body> Usadas para inserir mídia na página, como imagens e vídeos:

  • <img /> Insere uma imagem na página (É possível inserir fotos de arquivos .jpg ou .png). Ex: <img src="caminho/para/imagem.jpg" alt="Descrição da imagem" />
  • <video></video> Insere um vídeo na página. Ex: <video src="caminho/para/video.mp4" controls></video>
  • <audio></audio> Insere um áudio na página. Ex: <audio src="caminho/para/audio.mp3" controls></audio>
  • <iframe></iframe> Insere um mapa ou uma interface de outro site
  • <picture></picture> Usada para inserir imagens responsivas na página
  • <source /> Usada dentro da tag <picture> ou <video> para definir diferentes fontes de mídia
  • <figcaption></figcaption> Usada para adicionar uma legenda a uma imagem ou figura
  • <figure></figure> Usada para agrupar uma imagem e sua legenda
  • <embed /> Usada para incorporar conteúdo multimídia, como vídeos ou áudios, na página
  • <object></object> Usada para incorporar conteúdo externo, como arquivos PDF ou Flash, na página
  • <param /> Usada dentro da tag <object> para definir parâmetros para o conteúdo incorporado
  • <map></map> Usada para definir áreas clicáveis em uma imagem
  • <area /> Usada dentro da tag <map> para definir uma área clicável específica
  • <canvas></canvas> Usada para desenhar gráficos e animações na página usando JavaScript

Tags de links

São colocados dentro da tag <body> Usadas para criar links para outras páginas ou recursos:

  • <a></a> Cria um link. Ex: <a href="https://www.exemplo.com">Clique aqui</a>

Tags de estrutura

São colocados dentro da tag <body> Usadas para estruturar a página:

  • <header></header> Define o cabeçalho da página ou de uma seção
  • <main></main> Define o conteúdo principal da página
  • <footer></footer> Define o rodapé da página ou de uma seção
  • <div></div> Define uma seção genérica na página
  • <span></span> Define uma seção inline genérica na página
  • <section></section> Define uma seção temática na página
  • <nav></nav> Define uma seção de navegação na página
  • <aside></aside> Define um conteúdo lateral na página
  • <blockquote></blockquote> Define uma citação em bloco
  • <hr /> Insere uma linha horizontal para separar conteúdos

Tags de Back-End

São colocados dentro da tag <head> Usadas para integrar com linguagens de programação do lado do servidor ou do cliente:

  • <script></script> Usada para inserir código JavaScript na página ou para vincular arquivos JavaScript na página
  • <link /> Usada para vincular arquivos externos à página
  • <style></style> Usada para inserir estilos CSS diretamente na página
  • <meta /> Usada para definir metadados sobre o documento HTML

Tags de Listas

São colocados dentro da tag <body> Usadas para criar listas ordenadas e não ordenadas:

  • <ul></ul> Cria uma lista não ordenada (com marcadores)
  • <ol></ol> Cria uma lista ordenada (numerada)
  • <li></li> Define um item de lista

Tags de Tabelas

São colocados dentro da tag <body> Usadas para criar tabelas de dados:

  • <table></table> Cria uma tabela
  • <tr></tr> Define uma linha na tabela
  • <td></td> Define uma célula de dados na tabela
  • <th></th> Define uma célula de cabeçalho na tabela
  • <thead></thead> Define o cabeçalho da tabela
  • <tbody></tbody> Define o corpo da tabela
  • <tfoot></tfoot> Define o rodapé da tabela

Tags de formulários

São colocados dentro da tag <body> Usadas para criar formulários para entrada de dados:

  • <form></form> Cria um formulário
  • <input /> Cria um campo de entrada de dados
  • <textarea></textarea> Cria uma área de texto para entrada de dados
  • <button></button> Cria um botão
  • <select></select> Cria uma lista suspensa
  • <option></option> Define uma opção em uma lista suspensa
  • <label></label> Define um rótulo para um elemento de formulário

Atributos de elementos

Os atributos são usados para fornecer informações adicionais sobre um elemento HTML. Eles são colocados dentro da tag de abertura do elemento e geralmente vêm em pares nome-valor. Por exemplo, o atributo "href" é usado na tag <a> para especificar o URL do link:

                        <a href="https://www.exemplo.com">Clique aqui</a>

Alguns atributos comuns incluem:

  • id: Define um identificador único para um elemento
  • class: Define uma classe para um elemento, que pode ser usada para aplicar estilos CSS
  • src: Especifica o caminho para um recurso externo, como uma imagem ou um vídeo
  • alt: Fornece uma descrição alternativa para uma imagem, usada para acessibilidade
  • title: Fornece informações adicionais sobre um elemento, geralmente exibidas como uma dica de ferramenta
  • style: Permite aplicar estilos CSS diretamente a um elemento
  • width e height: Especificam a largura e a altura de um elemento, como uma imagem ou um vídeo
  • target: Especifica onde abrir o link (por exemplo, em uma nova aba)
  • name: Define o nome de um elemento, usado em formulários
  • value: Define o valor de um elemento, usado em formulários
  • placeholder: Fornece um texto de espaço reservado em campos de entrada
  • disabled: Desabilita um elemento, tornando-o não interativo
  • checked: Marca um elemento de entrada do tipo checkbox ou radio como selecionado
  • href: Especifica o URL de destino de um link
  • rel: Define a relação entre o documento atual e o recurso vinculado
  • type: Especifica o tipo de um elemento, como o tipo de entrada em um formulário
  • action: Define a URL para onde os dados do formulário serão enviados
  • method: Especifica o método HTTP a ser usado ao enviar o formulário (GET ou POST)
  • autocomplete: Habilita ou desabilita o preenchimento automático em campos de entrada
  • maxlength: Define o número máximo de caracteres permitidos em um campo de entrada
  • minlength: Define o número mínimo de caracteres exigidos em um campo de entrada
  • step: Define o incremento para campos de entrada numéricos
  • pattern: Define uma expressão regular que o valor de um campo de entrada deve corresponder
  • required: Torna um campo de entrada obrigatório
  • lang: Especifica o idioma do conteúdo de um elemento
  • charset: Especifica o conjunto de caracteres usado em um documento HTML
  • http-equiv: Fornece informações equivalentes a um cabeçalho HTTP
  • content: Usado com a tag <meta> para definir o valor do metadado
  • defer: Indica que um script deve ser executado após o documento ter sido analisado
  • async: Indica que um script deve ser executado de forma assíncrona
  • crossorigin: Define como os recursos devem ser compartilhados entre origens
  • integrity: Fornece uma verificação de integridade para recursos externos
  • referrerpolicy: Define a política de referência para links e recursos
  • sizes: Especifica os tamanhos de ícones para diferentes dispositivos
  • media: Define o tipo de mídia para o qual um recurso é destinado
  • hreflang: Especifica o idioma do recurso vinculado
  • sandbox: Aplica restrições de segurança a um iframe
  • allow: Define permissões para um iframe
  • loading: Especifica o comportamento de carregamento de imagens e iframes
  • novalidate: Desabilita a validação de formulários
  • autofocus: Define que um elemento deve receber foco automaticamente ao carregar a página
  • spellcheck: Habilita ou desabilita a verificação ortográfica em um elemento
  • contenteditable: Torna o conteúdo de um elemento editável pelo usuário
  • draggable: Define se um elemento pode ser arrastado
  • hidden: Oculta um elemento da visualização
  • tabindex: Define a ordem de tabulação para elementos interativos
