O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. Ele controla a aparência visual dos elementos na página, como cores, fontes, espaçamentos, layouts e até animações.

Como usar CSS?

Existem três maneiras principais de aplicar CSS a um documento HTML:

  • CSS Inline: Estilos aplicados diretamente aos elementos HTML usando o atributo "style".
  • CSS Interno: Estilos definidos dentro de uma tag <style> no cabeçalho do documento HTML.
  • CSS Externo: Estilos armazenados em um arquivo separado com a extensão .css, que é vinculado ao documento HTML usando a tag <link>.

Exemplo de CSS Externo

Veja um exemplo simples de como usar CSS externo:


                        <!DOCTYPE html>
                        <html lang="pt-BR">
                            <head>
                                <meta charset="UTF-8">
                                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                <title>Exemplo de CSS Externo</title>
                                <link rel="stylesheet" type="text/css" href="styles.css">
                            </head>
                            <body>
                                <h1>Olá, Mundo!</h1>
                                <p>Este é um exemplo de CSS externo.</p>
                            </body>                           
                        </html>

Conteúdo do arquivo styles.css


                            body {
                                background-color: #f0f0f0;
                                font-family: Arial, sans-serif;
                               }  
                            h1 {
                                color: #333333;
                                text-align: center;
                            }
                            p {
                                color: #666666;
                                font-size: 16px;
                            }

Como usar os comandos CSS?

Os comandos CSS são usados para definir estilos para elementos HTML. Eles são compostos por seletores e declarações. O seletor especifica o elemento HTML ao qual o estilo será aplicado, enquanto a declaração define as propriedades e valores do estilo.

Num comando CSS, existem 4 (mas na maioria dos usos, 3) partes principais do comando: o elemento, a propiedade do estilo, o valor da propiedade, e o operador.

Elemento

O elemento é o componente HTML ao qual o estilo será aplicado. Pode ser uma tag HTML, uma classe, um ID, ou todo o site no geral, é o primeiro componente do comando CSS:

  • Para colocar um estilo para um tipo de elemento html, o elemento deve ser declarado assim:

    
                        p :A {
                            Y: Z;
                        }

  • Para colocar um estilo para uma classe, o elemento deve ser declarado assim:

    
                        .nome-da-classe {
                            Y: Z;
                        }

  • Para colocar um estilo para um ID, o elemento deve ser declarado assim:

    
                        #nome-do-id {
                            Y: Z;
                        }

  • Se o estilo é para todo o site, o elemento deve ser declarado assim:

    
                        * {
                            Y: Z;
                        }

Às vezes o elemento para o estilo está dentro de outras tags e não é diretamente acessível, ou somente é necessário usar um fragmento dessa classe, então é necessário usar seletores mais específicos para aplicar o estilo corretamente, isso pode ser feito combinando seletores ou usando pseudo-classes:


                    div p {
                        Y: Z;
                    }
                    .caixa li{
                        Y: Z;
                    }

Às vezes o estilo precisa ser aplicado a mais de um tipo de elemento, porém, diante desse caso, em vez de criar dois comandos separados, é possível usar vírgulas para separar os seletores e assim ahorrar espaço na área de comandos:


                    h1, h2, h3 {
                        Y: Z;
                    }

Propriedade do estilo

A propriedade do estilo é a característica visual que você deseja modificar no elemento selecionado. Pode ser a cor do texto, o tamanho da fonte, o espaçamento entre linhas, a margem, o preenchimento, entre outros. A propriedade do estilo é especificada após o seletor e seguida por dois pontos (:).

Valor da propriedade

O valor da propriedade é o valor específico que você atribui à propriedade do estilo. Pode ser um valor numérico, um valor em pixels ou rems, um grupo de numeros, uma cor (em nome, hexadecimal, rgb, etc), uma URL, palavras-chave, etc. O valor da propriedade é especificado após os dois pontos (:) e seguido por um ponto e vírgula (;) As vezes, um elemento pode ter mais de uma propriedade de estilo, nesse caso, estes só precisam de um comando CSS, cada propriedade e valor devem ser separados por ponto e vírgula (;).

Operador

O operador é usado para poder aplicar o estilo ante uma interação com o usuario, seja um clique, seja que este tocando o cursor sobre o elemento, ou que o elemento esteja ativo, etc. O operador é colocado depois do elemento, antes da abertura das chaves ({), e é precedido por dois pontos (:).


                    a:hover {
                        color: blue;
                    }

Também é possível usar comentários (frases que aparecem apenas no código e não são exibidas no site) para documentar o código CSS e facilitar a compreensão:


                    /* Este é um comentário em CSS */
                    p {
                        color: red; /* Define a cor do texto como vermelho */
                    }
